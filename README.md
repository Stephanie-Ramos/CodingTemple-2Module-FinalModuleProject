# Personal Portfolio Website

## 📌 Overview
This project is a responsive personal portfolio website built using HTML and CSS. It is based on a portfolio template and showcases sections such as case studies, testimonials, recent work, and a contact form.

The goal of this project was to practice front-end web development skills, including layout design, responsive design, and mobile navigation.

---
## 🚀 Features

* Responsive design for desktop and mobile devices
* Fixed navigation bar with hamburger menu for mobile
* Smooth scrolling between sections
* Flexbox and CSS Grid layouts
* Interactive navigation menu using JavaScript
* Contact form layout
* Clean and modern UI design

---
## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* JavaScript (for mobile menu toggle)
* Font Awesome (icons)
* Google Fonts

---
## 📱 Responsive Design

The website is designed with a mobile-first approach and includes:

* Hamburger menu for smaller screens
* Flexible layouts using media queries
* Optimized images and content for different screen sizes

## 🎯 Key Sections

---
## ⚙️ How It Works
* The navigation bar includes anchor links that scroll smoothly to each section.
* A hamburger icon appears on mobile devices and toggles the menu using JavaScript.
* Layouts are built using Flexbox and Grid for responsiveness.
